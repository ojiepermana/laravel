# Release 1.1.0

Release date: 2026-02-20
Tag: `1.1.0`

## Highlights

- Added **BNI OGP H2H v2 Payment** module (Laravel-focused):
  - OAuth Token
  - In-house Inquiry
  - Interbank Inquiry
  - Do Payment
  - Get Payment Status
- Added Laravel integration for payment module:
  - Service provider
  - Facade (`bni-h2h`)
  - Contract binding
- Added support utilities and typed exceptions for payment flow:
  - JWT signature utility
  - Input sanitizer
  - Reference number generator
  - HTTP/API exception classes
- Added comprehensive tests for payment and utilities.

## BNI Configuration Unification

- Unified Billing + Payment configuration into one file: `config/bni.php`
- New config namespaces:
  - `bni.billing.*`
  - `bni.payment.*`
- Environment variables are now separated to avoid collisions:
  - `BNI_BILLING_*`
  - `BNI_PAYMENT_*`

## Billing Naming Refactor

- Introduced new Billing classes with clearer naming:
  - `BniBillingClient`
  - `BniBillingEncryptor`
- Migrated usages, tests, and docs to the new names.

## Breaking Changes

- Legacy Billing classes were removed:
  - `BNIAPIServices`
  - `BNIEncryptServices`
- If your code references legacy classes, migrate to:
  - `OjiePermana\Laravel\Bank\BNI\Billing\BniBillingClient`
  - `OjiePermana\Laravel\Bank\BNI\Billing\BniBillingEncryptor`
